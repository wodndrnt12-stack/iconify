# 콘텐츠 엔진

같은 내용을 모든 곳에 게시하는 대신, 하나의 아이디어를 강력한 플랫폼 네이티브 콘텐츠로 만드세요.

## 활성화 조건

- X 포스트 또는 스레드 작성 시
- LinkedIn 포스트 또는 런칭 업데이트 초안 작성 시
- 숏폼 비디오 또는 YouTube 설명 영상 스크립팅 시
- 아티클, 팟캐스트, 데모, 또는 문서를 소셜 콘텐츠로 재활용 시
- 런칭, 마일스톤, 또는 테마를 중심으로 경량 콘텐츠 계획 수립 시

## 첫 번째 질문

다음을 명확히 하세요:
- 소스 에셋: 무엇을 기반으로 조정할 것인가
- 대상: 빌더, 투자자, 고객, 운영자, 또는 일반 대중
- 플랫폼: X, LinkedIn, TikTok, YouTube, 뉴스레터, 또는 멀티 플랫폼
- 목표: 인지도, 전환, 채용, 권위 확립, 런칭 지원, 또는 참여도

## 핵심 규칙

1. 플랫폼에 맞게 조정하세요. 동일한 카피를 크로스 포스팅하지 않습니다.
2. 요약보다 훅이 더 중요합니다.
3. 모든 포스트는 하나의 명확한 아이디어를 담아야 합니다.
4. 슬로건보다 구체적인 내용을 사용하세요.
5. 행동 요청(CTA)을 작고 명확하게 유지하세요.

## 플랫폼별 가이드

### X
- 빠르게 시작하세요
- 포스트 또는 스레드의 트윗당 하나의 아이디어
- 꼭 필요한 경우가 아니면 본문에 링크를 포함하지 마세요
- 해시태그 남용 금지

### LinkedIn
- 강한 첫 문장
- 짧은 단락
- 교훈, 결과, 시사점에 대한 명시적 프레이밍

### TikTok / 숏폼 비디오
- 첫 3초가 주의를 사로잡아야 합니다
- 단순 나레이션이 아닌 비주얼 중심으로 스크립트 작성
- 하나의 데모, 하나의 주장, 하나의 CTA

### YouTube
- 결과를 일찍 보여주세요
- 챕터별로 구성하세요
- 20-30초마다 비주얼을 전환하세요

### 뉴스레터
- 관련 없는 항목의 묶음이 아닌 하나의 명확한 관점 전달
- 섹션 제목을 스캔하기 쉽게 만드세요
- 시작 단락이 실질적인 역할을 하도록 유지하세요

## 재활용 흐름

기본 캐스케이드:
1. 앵커 에셋: 아티클, 비디오, 데모, 메모, 또는 런칭 문서
2. 3-7개의 원자적 아이디어 추출
3. 플랫폼 네이티브 변형 작성
4. 출력물 간 반복 제거
5. 플랫폼 의도에 맞게 CTA 정렬

## 결과물

캠페인 요청 시 반환:
- 핵심 각도
- 플랫폼별 초안
- 선택적 게시 순서
- 선택적 CTA 변형
- 게시 전 필요한 누락 입력 사항

## 품질 기준

전달 전 확인:
- 각 초안이 해당 플랫폼에 네이티브하게 읽히는가
- 훅이 강하고 구체적인가
- 일반적인 홍보성 문구가 없는가
- 요청하지 않은 경우 플랫폼 간 중복된 카피가 없는가
- CTA가 콘텐츠 및 대상과 일치하는가
