# 훅 시스템

## 훅 타입

- **PreToolUse**: 도구 실행 전 (유효성 검사, 파라미터 수정)
- **PostToolUse**: 도구 실행 후 (자동 포맷, 검사)
- **Stop**: 세션 종료 시 (최종 확인)

## 자동 수락 권한

주의하여 사용:
- 신뢰할 수 있고 잘 정의된 계획에만 활성화
- 탐색적 작업에는 비활성화
- dangerously-skip-permissions 플래그 절대 사용 금지
- 대신 `~/.claude.json`의 `allowedTools`에서 설정

## TodoWrite 모범 사례

TodoWrite 도구 사용 목적:
- 다단계 태스크의 진행 상황 추적
- 지시 사항 이해 확인
- 실시간 조정 지원
- 세분화된 구현 단계 표시

할 일 목록을 통해 확인 가능한 항목:
- 순서가 잘못된 단계
- 누락된 항목
- 불필요한 추가 항목
- 잘못된 세분화 수준
- 잘못 해석된 요구사항
